## Swahili (sw)

### Data size: 1k
- Train  
  ```bash
  python train.py --lang sw --data_size 1k --src_lang_code swh_Latn
  ```

  ```bash
  python evaluate.py --lang sw --data_size 1k --src_lang_code swh_Latn --test_file_src ./test_data/flores200_sw.txt --test_file_ref ./test_data/flores200_en.txt
  ```
